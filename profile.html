About Me

I am an engineer with a strong focus on building secure, identity-driven applications across web, mobile, and API layers. I design and implement REST APIs and backend services that integrate with identity and access management systems such as Microsoft Entra ID and Okta, with a strong emphasis on secure authentication and authorisation patterns using OAuth2 and SAML. On the frontend, I build responsive web applications using React, and I also develop Android applications, ensuring consistent secure authentication flows across platforms. My work combines full-stack development with a security-first mindset, focusing on scalable architectures, clean API design, and robust identity integration.
